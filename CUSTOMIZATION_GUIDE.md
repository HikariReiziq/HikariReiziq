# 📚 Panduan Kustomisasi README Profile

## 🎨 Cara Mengedit Profil GitHub Anda

### 1. Header Banner
- **Lokasi**: Bagian atas file README.md
- **Edit**: Ganti `text=HikariReiziq` dengan nama Anda
- **Warna**: Ubah `customColorList=6,11,20` untuk warna yang berbeda
- **Animasi**: Ganti `animation=twinkling` dengan: `fadeIn`, `scaleIn`, `blink`

### 2. Badges Teknologi
- **Tambah badge baru**: Kunjungi [shields.io](https://shields.io)
- **Format**: `![Name](https://img.shields.io/badge/TEXT-COLOR?style=for-the-badge&logo=LOGO)`
- **Contoh**: `![React](https://img.shields.io/badge/React-61DAFB?style=for-the-badge&logo=react&logoColor=black)`

### 3. Social Media Links
- **Edit di bagian**: `<!-- Connect With Me -->`
- **Ganti URL**: Ubah `https://linkedin.com/in/hikarireiziq` dengan profil Anda
- **Tambah platform**: Salin format badge dan ganti logo/URL

### 4. Stats & Charts
- **GitHub Stats**: Otomatis menggunakan username GitHub
- **Tema**: Ganti `theme=tokyonight` dengan: `dark`, `radical`, `merko`, `gruvbox`
- **Languages**: Akan otomatis mendeteksi bahasa yang paling sering digunakan

### 5. Snake Animation
- **Setup**: GitHub Action akan berjalan otomatis setiap 24 jam
- **Manual**: Pergi ke Actions > Generate snake animation > Run workflow
- **Customization**: Edit file `.github/workflows/snake.yml`

### 6. Quote Footer
- **Edit quote**: Ganti bagian `"Security is not a product..."`
- **Author**: Ganti `- Bruce Schneier` dengan penulis quote Anda

## 🛠️ Tools yang Digunakan

1. **[Capsule Render](https://github.com/kyechan99/capsule-render)** - Header/Footer banner
2. **[GitHub Stats](https://github.com/anuraghazra/github-readme-stats)** - Statistics cards
3. **[Shields.io](https://shields.io)** - Badges dan icons
4. **[GitHub Streak](https://github.com/DenverCoder1/github-readme-streak-stats)** - Streak statistics
5. **[Snake Game](https://github.com/Platane/snk)** - Contribution animation
6. **[Profile Views](https://github.com/antonkomarev/github-profile-views-counter)** - View counter

## 🎯 Tips Kustomisasi

- **Konsistensi warna**: Pilih 2-3 warna utama dan gunakan konsisten
- **Jangan berlebihan**: Pilih elemen yang benar-benar penting
- **Update berkala**: Perbarui informasi "Currently Working On"
- **Test preview**: Gunakan VS Code atau editor dengan preview markdown
- **Mobile friendly**: Pastikan tampilan bagus di mobile
